# MQTT Timer WebSocket App
Uma aplicação Node.js que conecta MQTT com WebSocket para exibir um timer em tempo real na web.

## Funcionalidades
- Conexão MQTT para receber comandos remotos
- Timer controlado via mensagens MQTT
- WebSocket para comunicação em tempo real
- Interface web para visualizar o timer
- Suporte a Docker para deploy fácil

## Tecnologias
Node
MQTT
WebSocket - Comunicação bidirecional em tempo real
Docker - Containerização da aplicação
HTML/CSS/JS - Interface web simples
